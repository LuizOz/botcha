h1. Botcha Manual

Botcha is an python irc bot

h2. Requirements

Botcha uses some python libraries, so, on debian systems type:

<pre><code>apt-get install python-irclib python-beautifulsoup python-mechanize python-httplib2 sqlite3</code></pre>

h2. Installation

<pre><code>git clone https://github.com/but3k4/botcha
cd botcha
mkdir log
sqlite db/sqlite.db < db/schema.sql
vim conf/lokky.cfg
python bot.py</code></pre>
